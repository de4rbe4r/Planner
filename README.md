## Project name: Task Planner
### Description: Web application for project planning. In this application, it is possible to create projects with assigned responsibilities, create individual employees, create tasks and assign those responsible for the task. Each entity (project, task, employee) has the ability to edit
---

The list of employees displays information on each employee (first name, last name, position), as well as projects in which he is appointed responsible. Each entry can be edited and deleted. When an employee is removed, he is automatically removed from all projects in which he was appointed responsible.
<br/><br/>
<div align="center"><img src="https://github.com/de4rbe4r/Planner/blob/master/Images/2023-06-05_15-27-26.png" width="900"/></div>
</br>
When you click on the "Редактировать" button, all information about the entity is transferred to the modal window, where the data is specified. For example, you can add/remove responsible persons and the list of responsible persons for the project will be changed new for the project.
</br></br>
<div align="center"><img src="https://github.com/de4rbe4r/Planner/blob/master/Images/2023-06-05_15-28-13.png" width="700"/></div>
</br>
When creating a task for a project, you must first select the project to which this task belongs, then enter information about the task and assign a person responsible for the task. Under the filling form there is a table of tasks related to this project. Completed tasks are displayed as a gray bar and are located at the bottom of the list.
</br></br>
<div align="center"><img src="https://github.com/de4rbe4r/Planner/blob/master/Images/2023-06-05_15-28-32.png" width="900"/></div>
</br>
A search is implemented for each entity (employee, project, task). For an employee, search by various combinations of first name, last name and position, for a task and a project, search by title.
